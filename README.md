# Developing

Currently the project consists of two parts - main and renderer. The dependencies of both need to be installed separately.


```
# ./
npm i
cd ./renderer && npm i
```
### Start the frontend
```
# ./renderer
npm start
```
### Start the election app
```
npm run dev
```
